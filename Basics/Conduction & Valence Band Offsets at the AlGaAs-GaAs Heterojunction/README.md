Goal:
1. Using the bandgap difference between AlGaAs and GaAs (from Project 1),split the total bandgap difference into conduction band offset (dEc) and valence band offset (dEv) using the standard 60:40 band-offset
    rule commonly used for AlGaAs/GaAs.

Total bandgap difference:
    dEg(x) = Eg_AlGaAs(x) - Eg_GaAs

60:40 rule (widely used approximation for AlGaAs/GaAs, Dingle et al.):
    dEc = 0.6 * dEg
    dEv = 0.4 * dEg
