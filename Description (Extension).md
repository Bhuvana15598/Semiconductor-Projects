AlGaN/GaN HEMT Band Bending Model

Extension of a semiconductor device modeling project to visualize conduction band bending in an AlGaN/GaN HEMT heterostructure.

Features
1. Conduction band offset estimation
2. AlGaN/GaN heterojunction band diagram generation
3. Visualization of band bending across the AlGaN barrier
4. Triangular potential well formation at the interface
